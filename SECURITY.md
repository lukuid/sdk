# Security Policy

If you believe you have found a security issue in the LukuID SDK, please report it privately.

Examples include:

- verification bypasses
- trust profile confusion
- chain validation bugs
- canonicalization inconsistencies
- unsafe default trust behavior
- export behavior that weakens evidence integrity

Please contact:

info@lukuid.com

Do not publicly disclose vulnerabilities affecting production verification or trust behavior until coordinated remediation is available.

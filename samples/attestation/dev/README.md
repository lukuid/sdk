# Development Attestation Sample Certificates

These DER files are sample development attestation certificates for the LukuID
device chain:

- `attestation_dac_der.der`: device attestation certificate (DAC)
- `attestation_manufacturer_der.der`: manufacturer certificate
- `attestation_intermediate_der.der`: intermediate certificate

They are kept under `samples/` because they are reference/test material, not
runtime assets loaded by the SDK by filename.
